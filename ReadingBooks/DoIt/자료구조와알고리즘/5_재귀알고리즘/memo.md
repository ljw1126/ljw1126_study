## 5. 재귀 알고리즘 

### 5-1. 재귀 알고리즘의 기본 
#### 재귀(recursion)
> 어떠한 이벤트에서 자기 자신을 포함하고 다시 자기 자신을 사용하여 정의되는 경우 재귀(recursion) 라고 함 
```
1. 하향식 분석(top-down analysis)
   가장 위족에 위치한 상자의 함수 호출부터 시작하여 계단식으로 자세히 조사해 나가는 분석 방법 ( 위 -> 아래 -> 위)
2. 상향식 분석(bottom-up analysis)
   하향식과 반대로 아래쪽부터 쌓아 올리며 분석하는 방법 (아래 -> 위)
```

###### ※ math.factorial()함수 (p187) 
```
- 파이썬에서는 표준 라이브러리로 모듈에서 factorial()함수를 제공함 
- math.factorial(x)는 정수 x의 팩토리얼 값을 반환하는데, 
  x가 정수가 아니거나 음수라면 ValueError 예외처리를 내보냄 
```

###### 재귀호출(recursive call)
```
- 예제 factorial.py에서 factorial() 함수가 팩토리얼 값을 구하기 위해 자기자신을 호출함 
  > 이것을 재귀호출(recursive call)이라 함
```

###### 유클리드 호제법(Euclidean algorithm)
```
- 최대공약수(GCD , Greatest Common Divisor) 구하는 알고리즘 
- '두 개의 자연수에 대한 최대공약수'를 구하는 대표적인 알고리즘
   > 두 자연수 A,B 에 대하여 ( A > B ) A를 B로 나눈 나머지를 R이라고 합시다.
   > 이때 A와 B의 최대공약수는 B와 R의 최대공약
   > GCD(A, B) = GCD(B, A%B)
   > a%b가 0일때 b가 최대 공약수다
```
###### ※ math.gcd()함수 (p191)
```
- 파이썬에서는 최대 공약수를 구하는 표준 라이브러리로 math 모듈에서 gcd()함수를 제공함 
- math.gcd(a,b) 는 정수 a,b의 최대공약수 반환함 
- 두 인수가 모두 0인 경우 gcd(0,0)은 0을 반환함 
```
