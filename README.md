## VScode 단축키 
```
1. 인터페이스 상속시 오버라이드 함수 표시 하는 방법
    - 인터페이스 명 선택 후 crtl + . 
2. import 한번에 다 하기 
    - Alt + Shift + O
3. getter / setter 생성하기 
    - 해당 속성 드래그 해서 선택 
    - F1 또는 다른 단축키 
    - getter 검색하면 generator 나옴 
```
